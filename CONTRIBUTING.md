# Contributing

このリポジトリへの改善提案を歓迎します。

## 基本方針

- 小さく分かりやすい変更を歓迎します
- PowerShell の命名やコメントは読みやすさを優先してください
- API キーや実 URL などの秘密情報は含めないでください

## Pull Request の流れ

1. Issue で目的を共有
2. 小さな単位でブランチを作成
3. 変更内容と確認方法を README / PR 本文に記載
4. `-WhatIf` 実行例やログ例があれば添付

## 変更時の確認観点

- `.pdf / .txt / .csv` のみ処理されるか
- サブフォルダー構造が維持されるか
- `rename=false` の扱いが期待通りか
- タイムスタンプが保持されるか
- 同名衝突時に連番が付与されるか
- `-WhatIf` / `-Confirm` が壊れていないか
