# Benchmark Checklist

電子帳簿保存法向けの命名支援バッチを評価するためのチェックリストです。

## 抽出精度

- [ ] 取引日付を正しく抽出できる
- [ ] 取引先を正しく抽出できる
- [ ] 金額を正しく抽出できる
- [ ] 和暦を西暦へ正しく変換できる
- [ ] 小計 / 消費税 / 合計のうち、意図した金額を選べる
- [ ] 複数候補がある場合に `rename=false` に倒せる

## ファイル処理

- [ ] `.pdf / .txt / .csv` だけを処理する
- [ ] 拡張子の大文字小文字を区別しない
- [ ] 入力フォルダーのサブフォルダー構造を維持する
- [ ] 同名ファイル時に `_0001` 形式で衝突回避する
- [ ] タイムスタンプを保持する

## 運用性

- [ ] ログに API 生レスポンスが出る
- [ ] ログに抽出結果が出る
- [ ] エラー時に次ファイルへ進める
- [ ] `-WhatIf` で事前確認できる
- [ ] `CopyNonRenamedFiles` の挙動が期待通り
